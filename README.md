# 💳 Credit Scoring Model

## 📌 Overview

This project builds a **Machine Learning-based Credit Scoring System** that predicts whether an individual is a **Good Credit Risk** or **Bad Credit Risk** using past financial data.

---

## 🎯 Objective

To predict an individual’s **creditworthiness** using historical financial and behavioral features.

---

## ⚙️ Approach

This problem is treated as a **Binary Classification Task** in Machine Learning.

We trained and compared multiple models:

* Logistic Regression
* Decision Tree
* Random Forest

The best model is selected based on performance metrics.

---

## 📊 Features Used

* 💰 Income
* 💳 Debt
* 📈 Payment History
* 🏦 Loan Amount
* ⚙️ Debt-to-Income Ratio (Feature Engineering)

---

## 🧠 Model Evaluation Metrics

* ✔ Accuracy
* ✔ Precision
* ✔ Recall
* ✔ F1-Score
* ✔ ROC-AUC

---

## 📈 Results

* Compared multiple models
* Selected the best-performing model
* Visualized performance using:

  * Confusion Matrix
  * ROC Curve

---

## 🎯 Final Output

The model predicts:

* ✅ **Good Credit**
* ❌ **Bad Credit**
